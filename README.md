# GProgASM3
**Metroidvania Platformer Adventure**  
A classic 2D Metroidvania-style platformer inspired by Super Mario Bros and Castlevania, featuring non-linear exploration, ability unlocking, enemy combat, collectibles, and multiple interconnected areas.

### Requirements
- Unity **6000.2.10f1** (exact version used for this project)  
- 2D URP Template (highly recommended)  
- Git LFS (for large asset files)

### Installation
1. Clone or download this repository
2. Open Unity Hub ‚Üí **Add** ‚Üí Select the cloned folder.
3. Wait for Unity to import everything (may take a few minutes due to asset size).
4. Open scene: `Assets/Scenes/MainMenuScene.unity`
5. Press **Play** to start the game!

### Project Settings (Important)
Make sure these settings match in **Edit ‚Üí Project Settings**:
- **Color Space**: Linear
- **Rendering ‚Üí Universal Render Pipeline ‚Üí 2D Renderer**
- **Physics 2D ‚Üí Gravity**: (0, -9.81)

## üéÆ Gameplay
This is a **Metroidvania** game with Mario-like tight platforming controls combined with ability-gated progression.

### Core Features
- Precise 2D platforming with coyote time, jump buffering, and variable jump height
- Double jump, wall slide, wall jump, dash (unlocked progressively)
- Multiple biomes: Forest ‚Üí Cave ‚Üí Temple ‚Üí Sky Ruins
- Health & energy system, melee + ranged attacks
- Save/Load system with checkpoints
- Collectibles: Health Upgrades, Energy Shards, Secret Rooms
- Boss fights and mini-bosses
